# SHACK-SERVER Vision

## Mission

Collect.
Normalize.
Analyze.
Prioritize.
Display.

SHACK-SERVER helps radio amateurs identify the most relevant stations from multiple information sources.

The focus is not on displaying more information.

The focus is on displaying the **right information**.
